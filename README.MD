## 6. Rotas do projeto

Rotas globais em `loja/urls.py`:

- `/admin/` -> admin Django

- `/pedido/` -> app `pedido`

### 6.1. App `produto`

- `GET /` -> lista de produtos (`produto:lista`)

### 6.2. App `perfil`

- `GET|POST /perfil/` -> cadastro de usuĂˇrio/perfil (`perfil:perfil`)

- `GET|POST /perfil/login/` -> autenticaĂ§ĂŁo (`perfil:login`)

### 6.3. App `endereco`

- `GET|POST /endereco/` -> novo endereĂ§o (`endereco:inicio`)
- `GET|POST /endereco/novo/` -> novo endereĂ§o (`endereco:novo`)
- `GET|POST /endereco/editar/<pk>/` -> editar (`endereco:editar`)
- `GET|POST /endereco/excluir/<pk>/` -> excluir (`endereco:excluir`)

### 6.4. App `carrinho_de_compras`

- `GET /carrinho_de_compras/` -> visualiza carrinho (`carrinho_de_compras:carrinho`)
